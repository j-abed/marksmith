Core principles

Your words are yours.

Documents live in your browser. Autosave stays on your device — no account, no upload, no server reading your drafts.

Markdown is canonical.

The editor always holds plain .md text. Hybrid decorations soften syntax visually without rewriting your source.

Preview is derived.

Rendered HTML flows through rehype-sanitize before it reaches the DOM. Unsafe markup never becomes authoritative state.

Split mode

Edit and preview in parallel

Split mode keeps CodeMirror and a live preview side by side with proportional scroll sync in both directions.

  • Click preview blocks to jump the editor
  • GFM tables, task lists, and fenced code
  • Syntax highlighting in editor and preview

Raw & preview

Write raw. Read rendered.

Raw mode is a full CodeMirror 6 surface with formatting commands and keyboard shortcuts. Preview mode is read-only output you can trust.

  • Bold, links, lists, blockquotes, code blocks
  • Copy Markdown or sanitized HTML
  • Export .md anytime
